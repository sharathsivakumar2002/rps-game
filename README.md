# 🎮 Rock Paper Scissors – AI Game

## 📌 Overview

An interactive **Rock Paper Scissors game** where users play against an AI opponent.
The game features real-time score tracking, instant feedback, and a clean responsive UI.

---

## 🚀 Features

* 🤖 Play against AI (random strategy)
* 🎯 Real-time result display
* 📊 Score tracking system
* ⚡ Fast and responsive UI
* 🎮 Simple and interactive gameplay

---

## 🛠 Tech Stack

* HTML
* CSS
* JavaScript (Vanilla)

---

## ▶️ How to Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/sharathsivakumar2002/rps-game.git
   ```
2. Open `index.html` in your browser
